Before start you need to **Install Java**

### To start server:
```
java -jar out/artifacts/Server/server.jar 
``` 

### To start client:
```
java -jar out/artifacts/Client/client.jar 
``` 

In client you can use command for 
create Json object \
```/json @object```

```
JsonObject{
    name: String,
    age: Int,
    dotnet: String
}
```
##### Sample: 
```
{ name: "John", age: 34} or { name: Test, dotcom: "vk.com" }
```
